## ✨ Features

### � **Immersive Hero Experience**
- **Sequential Entrance Animation** - A carefully choreographed "one-by-one" reveal: Profile → Name → Title → Description → Buttons.
- **Interactive Particle Background** - Dynamic background that responds to mouse movement and clicks.
- **AI-Like Typing Effect** - Real-time character typing animation for the main description.
- **Pulse Profile** - Subtle breathing animation for the profile picture.

### 🎨 **Modern Design System**
- **Custom CSS3 Architecture** - Built from scratch with variables for theming (no heavy frameworks).
- **Glassmorphism** - Premium frosted glass effects on cards and navigation.
- **Vibrant Gradients** - Curated color palettes (Blue/Purple/Emerald/Amber).
- **Dark/Light Theme** - Seamless theme switching with localStorage persistence.

### � **Fully Responsive**
- **Mobile Optimized** - Touch-friendly interactions and layout adjustments.
- **Adaptive Layouts** - Smooth transitions between mobile, tablet, and desktop views.
- **Haptic Feedback** - Subtle vibration feedback on mobile interactions.

## 🚀 Quick Start

### 1️⃣ **Clone the Repository**
```bash
git clone https://github.com/DARKBOYJYOTI/darkboyjyoti.github.io.git
cd darkboyjyoti.github.io
```

### 2️⃣ **Open in Browser**
```bash
# Simply open index.html in your browser
# Or use a local server (recommended)
python -m http.server 8000
# Visit: http://localhost:8000
```
